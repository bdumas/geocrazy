GeoCrazy est un plugin pour Dotclear 2 qui permet de géolocaliser les billets d'un blog, c'est-à-dire associer des coordonnées géographiques à chaque billet du blog.
Cette géolocalisation permet :
  • de transformer les flux RSS et atom de votre blog en flux GeoRSS ;
  • d'afficher pour chaque billet une carte du lieu associé.

1. Configuration

1.1. Clé d'API Google Maps

GeoCrazy utilise l'API Google Maps pour générer des cartes. Pour cela, une clé différente est nécessaire pour chaque site. Vous pouvez demander gratuitement une clé à Google en vous rendant à l'adresse suivante : http://code.google.com/intl/fr/apis/maps/signup.html.
Lorsque vous avez récupérer votre clé Google Maps, rendez-vous dans l'onglet "Configuration", collez la clé dans le champ texte et cliquez sur "Enregistrer".

1.2. Mode

Vous pouvez choisir entre le mode simple et le mode avancé.
Avec le mode avancé, vous pouvez utiliser plusieurs widgets GeoCrazy (voir le paragraphe 3.2) grâce au champ supplémentaire "ID" (n'oubliez pas de mettre à jour les bandeaux dans la rubrique "Widgets de présentation"). C'est pour le moment la seule différence entre les deux modes.

2. Géolocalisation d'un billet

2.1. Associer un lieu à un billet

Créez un nouveau billet ou modifiez un billet existant. Dans la colonne de droite de la page d'édition, une zone "Lieu" présente un lien "Ajouter un lieu". Cliquez sur ce lien pour ouvrir la popup de sélection d'un lieu. Sélectionnez un lieu en le recherchant via le formulaire ou en cliquant sur la carte pour positionner le marqueur à l'endroit désiré. Lorsque le marqueur est placé au bon endroit, cliquez sur "Enregistrer" pour valider et revenir sur l'écran d'édition du billet. Une carte apparaît alors pour indiquer le lieu associé au billet. Enregistrez le billet pour valider la localisation du billet.

2.2. Modifier le lieu associé à un billet

Pour modifier le lieu d'un billet, il suffit de cliquer sur le lien "Modifier le lieu" situé sous la carte sur la page d'édition du billet et de procéder comme à l'étape précédente.

2.3. Supprimer le lieu associé à un billet

Pour supprimer le lieu associé à un billet, et ne pas le remplacer par un autre, cliquez sur le lien "Modifier le lieu" dans la page d'édition du billet puis cliquez sur le lien "Retirer de la carte" dans la popup. Validez le changement sur la popup et enregistrez le billet.

3. Le widget GeoCrazy

Le widget GeoCrazy permet d'afficher une carte du lieu associé à un billet. Attention, cette carte ne s'affiche que sur la page d'un billet (et pas sur la page d'accueil donc).

3.1. Paramétrage du widget

Dans la rubrique "Widgets de présentation" de l'administration du blog, faites glisser le widget "Carte GeoCrazy" à l'emplacement désiré. En cliquant sur le signe "+", vous pouvez modifier les paramètres suivants :
  • Titre : permet d'afficher un titre au-dessus de la carte (peut-être vide) ;
  • Largeur : largeur de la carte en pixels (si aucune valeur n'est précisée, la carte prendra la plus grande largeur possible) ;
  • Hauteur : hauteur de la carte en pixels (si aucune valeur n'est précisée, la carte aura une hauteur de 200 pixels) ;
  • Zoom : niveau de zoom sur la carte (0 pour afficher la Terre entière, 19 pour zoomer au maximum (attention, certaines zones ne sont pas affichables pour un zoom trop élevé)) ;
  • Type : type de la carte (relief, plan, satellite ou mixte) ;
  • ID (en mode avancé seulement) : nécessaire si vous placez plusieurs widgets GeoCrazy sur une page (voir le paragraphe suivant).

3.2. Utilisation de plusieurs widgets

Il est possible de placer plusieurs fois le widget GeoCrazy dans les bandeaux de navigation et d'extra du blog. Par exemple si vous souhaitez afficher une carte globale et une carte rapprochée du lieu concerné. Dans ce cas, pour des raisons techniques, vous devez affecter une valeur au champ "ID" différente pour chaque widget (le champ "ID" n'est présent qu'en mode avancé, voir le paragraphe 1.2).

4. Flux GeoRSS et sitemap géographique

GeoCrazy transforme les flux de votre blog (Atom et RSS) en flux GeoRSS. GeoRSS est une extension des formats de syndication qui permet d'associer des coordonnées géographiques àux entrées du flux. Les billets non localisés restent bien sûr présents dans les flux.

En plus d'être consultables dans un agrégateur classique, les billets de votre blog peuvent maintenant être affichés dans une carte. Copiez-collez donc l'URL de votre flux RSS dans le champ "recherche" de Google Maps pour essayer. Vos billets géolocalisés s'affichent sur la carte. Sympa non ? Et cette manipulation n'est pas limitée qu'à Google Maps, vous pouvez par exemple faire la même chose avec Live Maps (mais c'est un peu plus long, il faut passer par le menu "Collections").

Un autre avantage du format GeoRSS est l'indexation des contenus en fonction de critères géographiques. Les contenus peuvent alors être restitués lors d'une recherche locale. Pour aider les moteurs de recherche à indexer vos contenus géolocalisés, GeoCrazy génère un sitemap géographique (à l'adresse indiquée dans l'onglet "Configuration") que vous n'avez plus qu'à soumettre aux moteurs de recherche.
